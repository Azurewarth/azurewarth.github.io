express_react_antd快速构建

项目快速构建，用于快速开始一个react项目，使用的组件库为ant-design，方便快捷，组件种类多，适用性强。

  1. 安装快速搭建脚手架create-react-app。使用yarn包管理工具。

  2. 进入project-package目录下，安装依赖后可以直接启动项目。

  3. 使用express-generator快速构建node层服务框架。

  4. 进入my-app目录，可以直接启动node层服务。

  5. 将project-package放入my-app目录内，进入project-package目录下的config文件夹中修改webpack打包文件，将输出目录换成my-app目录下的public文件夹。webpack打包后，文件直接进入public文件夹，启动node服务后，监听此文件夹。

  6. my-app文件夹下，routes中是服务编写的主要位置。

  7. 适当修改目录结构，便于开发需求，如下。

目录如下

使用node的Express框架，bin中位执行文件。routes中是所有的转发设置。

进入根目录，npm install安装依赖。

project_package是开发react-antd项目目录

……

config是配置包，public是静态文件包，scripts是脚本包，src是开发目录。

public

public包是index页面和其他静态文件，比如logo。

src包括页面，路由等。

进入project_package使用npm install安装依赖。

dev： 在project_package目录下，yarn start，http://localhost:3000进行开发。

build： 在project_package目录下， npm run build， 会在根目录public中生成打包文件。

production： 打打包后，在根目录下， npm start， 开启node服务，监听http://localhost:3000。
