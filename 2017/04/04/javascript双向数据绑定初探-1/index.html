javascript中对象重要方法之Object.defineProperty()

要了解数据绑定，先要了解javascript中对Object的构造。

Object中，基础的就是属性，比如上文vue中data下中message就是一个属性。

Object中，有一个对属性的定义的重要方法，就是Object.defineProperty()。该方法接受三个参数。

  1. 属性归属的对象的变量名
  2. 需要给对象定义的属性的名称
  3. 接收一个对象，主要可配置如下
    1. value: // 属性的值
    2. writable: // 是否可写
    3. enumerable: // 是否可枚举
    4. configurable: // 是否可配置上述属性
