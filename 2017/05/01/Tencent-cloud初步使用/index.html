Tencent cloud初步使用

1. 登陆注册，领取或购买腾讯云服务 (https://www.qcloud.com/)

​ 购买或领取服务后，可以选择服务器操作系统。选择Ubuntu 14版 32位。

2. 选择云产品中的云服务器。

3. 可以看到服务器选项。首先创建秘钥。

​ 秘钥创建好了后，能够下载私钥文件，此时服务器已经可以运行。

​ 可以通过( windows cmd or mac terminal)直接登录云服务器。

​ user是登录用户名。选择其他操作系统可以直接用root登录，选择Ubuntu操作系统，使用用户名ubuntu登录。

4. 设置登录名以及密码直接登录

​ 直接通过点击登录进入命令行。

​ 使用用户名和密码登录。进入etc/ssh/文件夹。

​ 看到最后一个文件sshd_config。使用vim打开修改。

​ 找到 PasswordAuthentication 改为 yes。

​ 使用 ubuntu 和 密码直接远程登录。

5. 安装相关所需

​ ubuntu可以使用apt-get直接安装相关需要的软件，但是不建议这样做，因为安装的软件都不是最新版本，可能会带来不可预料的问题。

​ 建议使用 wget直接获得。

​ 比如安装nodejs，进入nodejs/dist。进入v7.9.0文件夹，拷贝相关路径。

​ 进入命令行，使用wget url直接下载。然后解压，将所有文件拷贝放入/usr/local/node中。
